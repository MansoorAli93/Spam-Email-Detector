# Spam Email Detector 

## Overview
This project is a machine learning based web application that detects whether an email message is Spam or Not Spam using TF-IDF vectorization and a Linear SVM classifier.

## Features
- Interactive web interface using Streamlit
- Real-time spam prediction
- TF-IDF text vectorization
- Linear SVM classification
- Model accuracy display

## Resources Used
- Python
- Streamlit
- Scikit-learn

  ## Purpose
Demonstration of basic Natural Language Processing and Machine Learning classification.
